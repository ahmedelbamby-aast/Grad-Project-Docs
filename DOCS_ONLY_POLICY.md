# Docs-Only Repository Policy

## Scope
This repository must contain documentation files only.

## Allowed File Types
- `.md`
- `.markdown`

## Enforcement Rules
- Do not add source code files, binaries, media assets, or build artifacts.
- Keep all new content as Markdown documentation.
- If a non-documentation file is added by mistake, remove it before merging.

## Current Compliance
As of this update, all tracked files in this repository are documentation file types.
